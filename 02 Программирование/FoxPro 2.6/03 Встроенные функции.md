### Коды символов

Кодовая страница ANSI windows 1251 (русская кириллица)

| Кодовый диапазон | Символы |
|-|-|
| `10` | LF (line feed, перевод строки, `\n`) |
| `13` | CR (caret return, возврат каретки, `\r`) |
| `>=33` | обычные отображаемые символы начинаются с 33 кода |
| `[65; 90]` | A-Z |
| `[97; 122]` | a-z |
| `[192 ;223]` | А-Я |
| `[224; 255]` | а-я |
